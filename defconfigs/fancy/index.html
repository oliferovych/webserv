About

This project is a web server implementation that aims to provide a robust and efficient solution for serving web content. It supports various features such as handling HTTP requests, serving static files, and executing CGI scripts. The server is designed to be lightweight and easy to configure, making it suitable for both development and production environments.

Features

  • HTTP/1.1 protocol support
  • Configuration file handling
  • GET, POST and DELETE request handling
  • CGI script execution
  • Handling multiple CGI
  • Cookie creation
  • Logging and error handling
